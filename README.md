# Time of day quotes
This respository contains a collection of quotes from books and poems that mention a specific time of day. They are based on the [Literary Clock project](https://www.literaryclock.com/) with various additions from me and my friends.

## Quote guidelines
* Exact times, e.g., `14:22`, `2:22am/pm`, `noon`, etc. are straightforward.
* If a 12 hour time indicates the time of day, it should be used for that entry. For example, `two in the morning` should be used for `02:00`, not `14:00`. Otherwise, it can be used for either regardless of the quote's context, preferring the entry with fewer quotes. 
* `x minutes before / after / to / past / etc. y` should be assigned to the time it is referring to, not `y`. For example, `two minutes before noon` corresponds to `11:58`. `Quarter` and `half` also fall into this category.
* Quotes that have multiple times within them may be assigned to all of them, though, to increase variety, the more quotes a time has, the more this is discouraged.
* Not all times have an unambiguous quote available for them (yet), so some of them are fudged a bit. In particular:
    * `almost / nearly / about /etc.` can be used to refer to a time a few minutes off from the actual time. For example, `about 2am` can refer to `1:59am`.

## Format guidelines
1. The time of day itself should be surrounded by `<b></b>` tags.
1. If present, the following should be included within the tags:
    1. AM, PM, A.M., P.M., etc.
    1. Morning, afternoon, evening, night
1. `o'clock` should be included within the tags.
1. `hours` should be included within the tags when referring to the time, e.g., `1400 hours`.
1. Punctuation at the beginning or end of a time-of-day should not be included within the tags unless it comes from `A.M. / P.M.`.
1. Straight quotes (`'` and `"`) should be used over curly quotes (`’` and `”`). 
1. Double quotes should be used for dialogue, scare quotes, etc.
1. If a quote containing dialogue cuts off the full text and leaves a dangling quotation mark, the opening / closing mark should be added.
1. Use three periods instead of the ellipse character.
1. Avoid leading or trailing ellipses if they are not actually part of the quote. Instead, just start or end the quote as if it beigns or ends there.
1. Do not place spaces on either side of an ellipse.
    1. However, if the ellipse ends a sentence, a space should be added after it.
